# 小黄雀商单 B2B 版本开发汇报

## 一、今日开发目标

本次迭代的核心目标，是把原来偏“达人/品牌双视角 Demo”的产品，调整成更明确的 toB 商家服务模式：

- 商家是主要服务对象，可以登录后台、发布商单、查看报名候选人。

- 达人端不强制注册，降低参与门槛，可以直接浏览商单、查看详情、填写微信和账号链接报名。

- 前台页面以“透明商单广场”和“达人身价评估”为核心，弱化旧 Demo 感，便于对客户演示。

- 商单信息可复制成适合微信群、小红书私信转发的招募文案。

## 二、今日已完成内容

### 1. 产品定位与页面结构调整

- 产品名称统一为“小黄雀商单”，移除主要用户路径中的 Demo 表达。

- 前台导航调整为：

  - 首页

  - 商单广场

  - 商家登录 / 商家后台

- 新增正式产品路由：

  - `/`：首页，展示达人信息、达人当前身价、匹配商单。

  - `/deals`：公开商单广场。

  - `/deals/[id]`：商单详情和达人报名页。

  - `/merchant`：商家登录和商家工作台。

### 2. 首页体验优化

- 保留用户要求的“达人基础信息”滚动卡片，不删除、不改变核心交互。

- 保留“达人当前身价”模块，并与左侧达人卡片/匹配商单区域做布局对齐。

- 首页下方不再大面积展示完整商单广场，商单广场独立放到“商单广场”页面。

- 匹配商单区域改为只展示一个匹配商单，减少页面纵向冗余和右侧空白。

- 优化首页左右两栏高度关系，让页面更紧凑，减少明显空白区域。

### 3. 达人报价逻辑调整

- 对搜索出来的达人，改用后续确认的低价锚定公式：

  - 基础锚点：粉丝数 / 100

  - 报价区间：粉丝数 / 100 \* 0.8 到 粉丝数 / 100 \* 2

- 报价拆分为四个组成部分：

  - 粉丝基础价

  - 流量加成

  - 互动加成

  - 赛道标签加成

- 搜索达人使用低价锚定策略，避免把素人或小达人报价抬得过高。

- 首页预置达人可以保留更适合展示的报价效果，搜索结果则按更保守的商家友好公式计算。

### 4. 商单广场功能

- 新增独立商单广场页面，支持：

  - 商单列表展示

  - 搜索关键词过滤

  - 赛道筛选

  - 城市筛选

  - 查看商单详情

  - 复制分享文案

- 商单卡片展示信息包括：

  - 城市

  - 来源：平台精选 / 商家发布

  - 品牌

  - 商单标题

  - 目标赛道

  - 预算区间

  - 交付要求

  - 目标人群

  - 报名人数

### 5. 商单详情与达人报名

- 新增商单详情页，达人可以打开单独链接查看商单。

- 详情页展示：

  - 品牌和商单标题

  - 预算区间

  - 城市

  - 报名人数

  - 目标赛道

  - 交付要求

  - 目标人群

  - 详细说明

- 达人报名无需注册，只需要填写：

  - 微信号

  - 小红书/抖音主页链接

  - 昵称（选填）

  - 期望报价（选填）

  - 备注（选填）

- 已做重复报名限制：同一个微信和账号链接不能对同一个商单重复报名。

### 6. 分享复制文案优化

- 商单广场和详情页均支持复制适合转发的文案。

- 文案内容包含：

  - 小黄雀商单达人招募标题

  - 城市、品牌、商单标题

  - 预算区间

  - 适合达人

  - 赛道

  - 交付要求

  - 补充说明

  - 申请链接

  - 报名方式

- 已按最新要求处理：复制文案中不会出现 `运营微信：xiaohuangque`。

- 当商单联系微信是默认值 `xiaohuangque` 时，文案会自动改成“报名方式：打开链接填写微信和账号主页，无需注册”。

### 7. 商家登录与商家后台

- 新增商家登录入口。

- 当前 MVP 使用固定演示账号：

  - 账号：`admin`

  - 密码：`xiaohuangque2026`

- 登录后导航显示“商家后台”和“退出”。

- 商家后台支持：

  - 查看当前商家展示名

  - 修改商家展示名

  - 发布商单

  - 查看当前商家的商单

  - 按商单展开查看报名候选人

  - 修改候选人状态

### 8. 商家发布商单优化

- 商家后台新增商单发布表单。

- 支持字段：

  - 品牌 / 门店 / 产品

  - 商单标题

  - 城市

  - 最低预算

  - 最高预算

  - 目标赛道

  - 目标达人 / 人群

  - 交付要求

  - 详细说明 / 转发补充

  - 运营联系微信

* 新增目标人群和交付要求快捷选项。

* 表单校验已优化：品牌、标题、目标达人/人群、交付要求、预算区间必须合理。

### 9. 商家候选人管理

- 商家后台按商单展开候选人，而不是混在一个总列表里。

- 每个商单显示：

  - 品牌

  - 商单标题

  - 预算

  - 报名人数

  - 商单详情链接

- 展开后展示报名达人信息：

  - 昵称

  - 账号链接

  - 微信

  - 期望报价

  - 备注

  - 当前状态

- 候选人状态支持：

  - 待联系

  - 已选中

  - 不合适

  - `MarketplaceDeal`

  - `DealApplication`

  - `MerchantProfile`

- 新增商单 API：

  - 提交达人报名

  - 获取报名列表

- 后端启动时会初始化商单相关表。

- 已加入初始种子商单，保证商单广场启动后有内容可展示。

## 三、当前可演示路径

### 前台体验

1. 打开首页 `/`
2. 查看达人基础信息、达人当前身价和匹配商单
3. 点击“商单广场”
4. 浏览商单、搜索/筛选商单
5. 点击“查看并申请”
6. 在商单详情页填写微信和账号链接，提交报名

### 商家体验

1. 点击“商家登录”
2. 输入演示账号 `admin / xiaohuangque2026`
3. 进入商家后台
4. 发布一个新商单
5. 回到商单广场查看新商单是否出现
6. 达人报名后，商家后台可以按商单查看候选人
7. 商家可以修改候选人状态
